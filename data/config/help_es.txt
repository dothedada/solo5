Solo5 - Gestor de tareas para la terminal

Uso: 
  comando [opciones]

Comandos disponibles:
  hoy                    Cambia el contexto a Hoy (las tareas para hoy)
  global                 Cambia el contexto a Global (Todas las tareas)
  terminadas             Cambia el contexto a Terminadas (todas las tareas terminadas)
  ver (v)                Muestra las tareas en el contexto actual (Global, Hoy, Terminadas).
  agregar (a)            Añade una o más tareas. Separa múltiples tareas con "//".
  actualizar (act)       Edita una tarea existente. Solo una tarea a la vez.
  realizada (r)          Marca una tarea como completada (no reversible).
  borrar                 Elimina una tarea (no reversible).
  hacer                  Genera una lista de hasta 5 tareas para hoy.
  extra                  Añade más tareas al día después de completar las 5 iniciales.
  mañana                 Muestra las tareas pendientes para el próximo día.
  buscar (b)             Busca tareas que contengan una palabra clave.
  limpiar (l)            Limpia la selección actual de tareas.
  guardar (g)            Guarda el estado actual de todas las tareas.
  salir                  Sale de la acción actual o cierra la aplicación.
  purgar                 Elimina tareas completadas de hace más de 30 días.
  arreglar               Recalcula las fechas de vencimiento de las tareas.
  configurar (config)    Inicia la secuencia de configuración.
  ayuda                  Muestra esta ayuda.

Ejemplos básicos:
  agregar > Comprar leche hoy // Correr 3 kilómetros el viernes, muy difícil

Opciones avanzadas:
  - Usa "//" para agregar múltiples tareas en un solo comando.
  - Escribe fechas de forma natural (ej: "el próximo viernes").
  - Añade "!3" para asignar dificultad o "*" para marcar como inamovible.

Para más detalles, consulta la documentación completa.
